Esta versión de SammNew ajusta el servicio de actividades, mejora la navegación hacia los reportes técnicos y habilita el manejo de autenticación para la conexión de SAMM con Power BI.

<!-- truncate -->

## Novedades

- Se ajustó el servicio de actividades para que reciba el identificador de programación (idProgramación). ([SO-665](https://softwaresamm.atlassian.net/browse/SO-665))
- Se mejoró la navegación del botón "Nuevo" del reporte técnico hacia el formulario dinámico desde la pestaña de trabajos. ([SO-590](https://softwaresamm.atlassian.net/browse/SO-590))
- Se implementó el manejo de autenticación para la conexión de SAMM con Power BI. ([SO-614](https://softwaresamm.atlassian.net/browse/SO-614))
