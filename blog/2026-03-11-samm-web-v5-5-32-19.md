La versión **5.5.32.19** de SAMM Web trae mejoras de estabilidad en la ejecución de solicitudes web y en la gestión de permisos de equipos.

<!-- truncate -->

## Novedades

- Se solucionó un error que impedía completar la ejecución de solicitudes web cuando el campo de direcciones llegaba vacío, evitando así fallos inesperados en la aplicación. ([SO-473](https://softwaresamm.atlassian.net/browse/SO-473))
- Se ajustó la gestión de permisos de equipos para que el permiso "Cambiar estado equipo" funcione de manera independiente del permiso "Editar equipos", permitiendo cambiar el estado de un equipo aunque el rol no tenga habilitada su edición. ([SO-511](https://softwaresamm.atlassian.net/browse/SO-511))

## Correcciones

- Se incorporaron columnas de texto enriquecido en la capa de datos de SAMM Web, como base técnica para futuras mejoras de edición de contenido. ([SO-525](https://softwaresamm.atlassian.net/browse/SO-525))
