La versión **1.2.27.0** de SAMM API suma la gestión de certificados y de autenticación, una nueva sección de actividades, y corrige el manejo de fechas al crear eventos.

<!-- truncate -->

## Novedades

- Las líneas de requerimiento ahora aparecen enumeradas, facilitando su identificación y seguimiento. ([SO-640](https://softwaresamm.atlassian.net/browse/SO-640))
- Se agregó el CRUD para configurar certificados desde la matriz. ([SO-709](https://softwaresamm.atlassian.net/browse/SO-709))
- Se incorporó el manejo de autenticación como parte de las funcionalidades del servicio. ([SO-697](https://softwaresamm.atlassian.net/browse/SO-697))
- Se agregó una nueva sección de actividades. ([SO-714](https://softwaresamm.atlassian.net/browse/SO-714))
- Se creó un procedimiento almacenado que, ante la ocurrencia de eventos, ejecuta la generación automática de solicitudes de hallazgos. ([SO-553](https://softwaresamm.atlassian.net/browse/SO-553))

## Correcciones

- Se ajustó el manejo de fechas al crear eventos, evitando problemas relacionados con el ciclo del equipo. ([SO-662](https://softwaresamm.atlassian.net/browse/SO-662))
