La versión **1.2.25.0** de SAMM API ajusta el servicio de actividades e incorpora mejoras en la autenticación para la integración con Power BI.

<!-- truncate -->

## Novedades

- Se ajustó el servicio de actividades para que ahora reciba el identificador de programación (idProgramación). ([SO-665](https://softwaresamm.atlassian.net/browse/SO-665))
- Se mejoró el manejo de autenticación utilizado en la conexión de SAMM con Power BI. ([SO-614](https://softwaresamm.atlassian.net/browse/SO-614))

## Versiones requeridas

| Componente | Versión |
| ---------- | ------- |
| SAMMAPI    | 1.2.25.0 |
| SAMMLOGICA | 5.6.25.0 |
| CAPADATOS  | 2.1.10.3 |
| SAMCORE    | 2.0.20.0 |
| Base de Datos | C2.1.10.3 |
